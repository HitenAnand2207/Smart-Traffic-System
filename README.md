# Smart Traffic Analytics System

This project is an AI-powered traffic monitoring and analysis system using YOLOv8, OpenCV, and Streamlit.

## Features
- Real-time Vehicle Detection & Tracking
- Risk & Safety Scoring
- Pollution & Eco-efficiency Estimation
- Signal Recommendation (Advisory)
- Violation Logging (Red Light, Wrong Way)

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the dashboard:
   ```bash
   streamlit run app.py
   ```
